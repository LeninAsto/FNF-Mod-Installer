Quick Start

Up and running in 3 steps

  1. Enable your engines

    Open the Settings (⚙️) screen and toggle on the engines you use.

  2. Link the mods folder

    Tap Link Now on each engine tab and select the mods folder in the picker.

  3. Install a mod

    Share a ZIP file from any app, or use Add Mod inside the app, select your engine and confirm.

Good to know

Storage types explained

📁 Scoped Storage

Used by FNF Mobile, Plus Engine, P-Slice (Scoped). The mods folder lives inside Android/data/…. Open the game once first so the folder is created.

💾 Root Storage

Used by Psych Engine, NovaFlare, P-Slice (External). Folders like .PsychEngine sit in the root of your internal storage and require All Files Access permission.
